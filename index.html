🏥 Hospital DevOps Pipeline & API

Solução completa de Infraestrutura como Código e Automação de Dados Hospitalares.

🚀 Funcionalidades do Projeto

  • 🛡️ Segurança (DevSecOps): Validação automática de scripts SQL em busca de comandos perigosos.
  • 📊 Engenharia de Dados: Geração de massa de dados sintéticos em conformidade com a LGPD.
  • 📝 Auditoria: Mapeamento de artefatos e geração de audit_log.txt.
  • 🩺 Monitoramento: Dashboard de saúde do sistema via telemetria.
  • 🌐 Microserviço: API Flask expondo o status do sistema via JSON.

⚙️ Como Executar

Passo 1: Construir a Imagem
docker build -t hospital-devops-api .
Passo 2: Rodar o Container
docker run -p 5000:5000 hospital-devops-api

Acesse no navegador: http://localhost:5000

🛠️ Tecnologias

Docker Python 3 Flask SQL GitHub Actions
